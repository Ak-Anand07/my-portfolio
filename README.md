# My Portfolio

A simple personal portfolio website showcasing my skills, projects, and contact information. Built to highlight my work as a frontend developer and provide an easy way for recruiters to explore my projects.

## Live Demo
https://ak-anand07.github.io/my-portfolio/

## About
This portfolio is designed to give a quick overview of:
- Who I am  
- My technical skills  
- My featured projects  
- How to contact me  

The design is clean, responsive, and easy to navigate.

## Features
- Responsive personal portfolio
- About Me section
- Skills section
- Projects showcase
- Contact/social links
- Smooth UI with simple design

## Tech Used
- HTML  
- CSS  
- JavaScript  
- GitHub Pages (for deployment)
